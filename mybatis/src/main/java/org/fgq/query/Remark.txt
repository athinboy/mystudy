2、字段功能：
    (1)具有annotation.predicate注解，作为where条件字段。
    (2)没有annotation.predicate注解，值为null时不作为where条件。值不为null时作为where条件。
    (3)具有annotation.predicate注解，不作为insert操作的值列。
    (4)具有annotation.predicate注解，不作为update操作的值列。
    (5)具有annotation.predicate注解，不作为select操作结果列。
